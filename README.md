# Rock-Paper-Scissor-Classifier
Rock Paper Scissor Classifier using TensorFlow , Keras, Scikit Learn

DataSet: https://www.kaggle.com/frtgnn/rock-paper-scissor/

Layers:
 - Conv2D (64 filters)
 - MaxPolling
 - Conv2D (32 filters)
 - MaxPolling
 - Conv2D (32 filters)
 - MaxPolling
 - Flatten
 - DropOut(0.5)
 - Dense (128 neurons)
 - DropOut(0.5)
 - Dense (3 neurons)
